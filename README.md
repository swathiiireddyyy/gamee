# Neon Trail

A browser-based arcade game inspired by Qix-style territory capture. Move around the play area, create trails, capture territory, and avoid the enemies.

## Play locally

No build step or dependencies are required.

1. Open `index.html` in a modern web browser.
2. Select **START GAME**.
3. Use the arrow keys or swipe on a touch device to move.

The game loads the Press Start 2P font from Google Fonts when an internet connection is available.

## Project files

- `index.html` - game page and interface
- `style.css` - neon arcade styling
- `game.js` - canvas rendering, controls, enemies, scoring, and territory capture

## Deploy

This is a static web project and can be deployed with GitHub Pages, Netlify, Vercel, or any static hosting service. The entry point is `index.html`.
